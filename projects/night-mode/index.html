night-mode

Description

A lightweight extensible python script that quickly sets the backlight of your pc, opens certain programs (coming soon), and sets a sleep/shutdown timer for you

Details

CLI options are:

  • -b: Backlight to set, default 1
  • -c: Close action, eg displayoff, shutdown, suspend
  • -t: Time until close action is taken, eg 30m, 2.5h, default seconds
